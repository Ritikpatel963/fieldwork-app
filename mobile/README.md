# Mobile App

This is the React Native mobile app scaffold for AMPL Fieldwork.

## Setup

```bash
cd mobile
npm install
npx expo start
```

## Notes
- This uses Expo + TypeScript for Android and iOS development.
- Bootstrap is not directly supported in React Native; the app uses `NativeBase` for native-style components.
- Replace `src/screens/HomeScreen.tsx` and `src/screens/LoginScreen.tsx` with your Figma-designed UI.
